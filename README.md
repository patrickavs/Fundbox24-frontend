### App starten:

```bash
$ npm start
```

Sobald Metro gestartet ist, die `a`-Taste drücken,
um die App auf dem Android-Emulator, oder die `i`-Taste drücken, um die App auf einem iOS-Simulator zu starten.

### Tests auszuführen:

- ohne Test-Abdeckung

```bash
$ npm test
```

- mit Test-Abdeckung

```bash
$ npm run test:ci
```
